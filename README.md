# Socio
## A social networking site under development
### Requirement : 
  1. NodeJS
  2. MongDB
  3. Gulp
  4. Redis Server
### Project Setup :
  Install all packages and dependencies
  ```bash
    npm install
  ```
  Compress the asset files
  ```bash
    gulp build
  ```  
  Start the server
 ```bash
    npm start
  ```
