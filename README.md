# Socio
## A social networking site (Check this out: http://www.socio.codes/)
### Requirement : 
  1. NodeJS
  2. MongDB
  3. Gulp
  4. Redis Server
### Project Setup :
  Install all packages and dependencies
  ```bash
    npm install
  ```
  Run the server in development mode
 ```bash
    npm run dev
  ```
